  • Bug 40 - Hide Homepage Elements show/hide buttons can't use the "hidden" attribute anymore.
    - - Firefox 4 is adding support for a new attribute on elements called "hidden". When that attribute is present on an element, that element is... hidden.
    RTSE was already using a "hidden" attribute on the Show/Hide button in the Hide Homepage Elements feature, but for an unrelated purpose. To make sure everything still works for both Fx 3.7 and Fx4, this patch changes RTSE's "hidden" attribute to "rtsehidden".
