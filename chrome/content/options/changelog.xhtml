All changes are by KWierso unless otherwise specified.

  • Bug 50 - ATE breaking posts with [IMG] tags
    In recent Minefield (Firefox 4) builds, the ATE started handling strings of text differently, making some regular expression operations destroy post contents if the post had BBCode tags in uppercase letters.
    The good news is that those operations haven't been needed for at least a year, now, as the site itself can handle those uppercase tags just fine. This update removes those operations, fixing the problem.
