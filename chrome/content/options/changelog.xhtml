  • Bug 26 - Remove calls to extension manager in creating the smilies file on init
    - - With the new extension manager's API landing on the trunk, we need to remove a few calls to the old API.
    This patch changes the way the Smilies service is initialized so that it doesn't use the old API anymore.

    A side effect of this, though, is that you can't run RTSE from outside of the profile directory for dev work anymore.
    I hope to fix this sometime in the future, though.
