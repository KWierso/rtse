&Changelog.title;

07-28-2010

Go to http://bugzilla.roosterteethsiteextender.com for more information on each bug.

All changes are by KWierso unless otherwise specified.

  • Bug 38 - Remove useragent tweak from RTSE.
    - - Firefox is going to remove the ability to easily modify the useragent string. That'll either break RTSE (because we tweak the UA string), or it's just useless code (because it won't work). Either way, we should just not try to tweak the UA.
  • Bug 39 - Mod breakdown's have disappeared.
    - - The site is no longer displaying the mod breakdown stuff on profiles and homepages, due to performance issues. With that missing, RTSE's hide homepage elements feature is broken. This update changes the function to register a different element for handling the click.
