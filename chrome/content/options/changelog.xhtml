  • Bug 12 - Hide Homepage Breakdown not correctly initializing.
    - - This is a small fix to make sure that RTSE is initializing the breakdown hiding button's attributes. Currently, if you had the breakdown element hidden, the first time you click the button, nothing will happen, as RTSE doesn't know that it's already hidden.
    This update let's RTSE know about it, so the first click on the button will show the breakdown section.
